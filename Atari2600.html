This is a WIP cycle-accurate Atari2600 emulator. I saw Sethbling's Atari Emulator in Minecraft and was inspired to write my own emulator (but in Java, not Minecraft). I started this project in Sept 2020, but I put it on hold when school started. I plan on finalizing it during winter break.

So far, the CPU has been completely written, although it's far from bug-free. The TIA has the background, playfield, and player registers implemented, but there are still some option registers that need to be added. There is no user input or sound.
